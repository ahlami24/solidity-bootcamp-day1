# SimpleStorage Smart Contract
 Overview
This smart contract allows you to store, retrieve, and increment a number on the blockchain.

---

##  Functions

### store(uint256 _number)
Stores a number in the contract.

### retrieve()
Returns the stored number.

### increment()
Increases the stored number by 1.


##  How to Use

1. Deploy the contract in Remix
2. Call store() and enter a number (e.g., 10)
3. Call retrieve() to see the stored number
4. Call increment() to increase the number by 1
